# README

Displays the top 10 "hawt reads" from my other app "Url-lock-box"<br>

located at https://url-locken-boxen.herokuapp.com <br>

## Setup

``` $ git clone https://github.com/scmountain/Hawt-reads-.git```<br>
``` $ bundle ```<br>
``` $ rake db:create db:migrate ```<br>
